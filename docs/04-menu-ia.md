# 04. 메뉴 IA 및 화면 매핑

## 1단계 메뉴
- 대시보드
- 키워드 관리
- 수집
- 라벨링
- 글 작성
- 발행
- 로그/모니터링

## 2단계 메뉴 (현재 기준)
- 대시보드: 통합 현황, 단계별 상태
- 키워드 관리: 단일 메뉴(서브메뉴 없음)
- 수집: 수집 설정, 수집 실행, 작업 이력, 수집 데이터
- 라벨링: 라벨링 설정, 라벨링 실행, 라벨링 결과
- 글 작성: 글 작성 설정, 글 작성 실행, 페르소나 관리, 템플릿 관리, AI API 관리, 작성 결과/에디터
- 발행: 발행 설정, 발행 실행, 발행 채널 관리, 발행 이력
- 로그/모니터링: 실행 로그, 실패 로그, 재시도 큐

## 정책
- 수집/라벨링/글작성/발행 그룹은 서브메뉴 첫 항목을 항상 `설정`으로 유지
- 신규 서브메뉴 추가 시 설정 우선 순서가 깨지지 않도록 검토
