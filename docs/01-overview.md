# 01. 프로젝트 개요

## 목적
BlogWriter는 수집 -> 라벨링 -> 글 작성 -> 발행 파이프라인을 단일 데스크톱 앱에서 운영하기 위한 도구입니다.

## 현재 상태(2026-03-01)
- 소스 구조 평탄화 완료: `src/blogwriter` 제거
- 메뉴 구조 재편 완료: 1단계(도메인) + 2단계(서브메뉴)
- 설정 우선 정책 적용: 수집/라벨링/글작성/발행의 서브메뉴 첫 항목은 `설정`
- 웹셸 404 이슈 해결: 런타임 assets 직접 서빙

## 핵심 모듈
- 수집: `src/collector`
- 라벨링: `src/labeling`
- 글작성: `src/writer`
- 발행: `src/publisher`
- 저장소: `src/storage`
- UI: `src/ui`
- 공통: `src/core`
