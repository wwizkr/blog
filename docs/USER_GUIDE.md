# MubloOps 사용설명서 (UI 기준)

작성일: 2026-03-01  
버전: 평탄화 구조 기준

## 1. 목적
이 문서는 현재 구현된 MubloOps UI의 메뉴 구성과 사용 순서를 빠르게 이해하기 위한 운영 가이드입니다.

## 2. 실행
1. 의존성 설치
   - `pip install -r requirements.txt`
2. 실행
   - `set PYTHONPATH=src`
   - `python run.py`

## 3. 메뉴 구성(현재)
아코디언 1단계 메뉴 + 2단계 서브 메뉴 구조입니다.

1. 대시보드
- 통합 현황
- 단계별 상태

2. 키워드 관리
- 카테고리/키워드
- 연관 키워드
- 차단 키워드

3. 수집
- 수집 설정
- 수집 실행
- 작업 이력
- 수집 데이터

4. 라벨링
- 라벨링 설정
- 라벨링 실행
- 라벨링 결과

5. 글 작성
- 글 작성 설정
- 글 작성 실행
- 페르소나 관리
- 템플릿 관리
- AI API 관리
- 작성 결과/에디터

6. 발행
- 발행 설정
- 발행 실행
- 발행 채널 관리
- 발행 이력

7. 로그/모니터링
- 실행 로그
- 실패 로그
- 재시도 큐

## 4. 데이터/저장 위치
- 기본 DB: `mublo_ops_data/mublo_ops.db`
- 기본 웹에디터 런타임/업로드: `mublo_ops_data/web_editor_runtime`

## 5. 권장 운영 순서
1. 키워드 관리에서 카테고리/키워드 정리
2. 수집 설정 점검 후 수집 실행
3. 수집 데이터/라벨링 결과 검토
4. 글 작성 설정 점검 후 글 생성
5. 발행 설정 점검 후 발행 실행
